Android 上的 Chrome：Kiwi 浏览器的调整和插件分享

引言

在 Android 上使用 Kiwi 浏览器已经有 2 周了。所以写一篇关于 Kiwi 浏览器的使用体验。相比手机上的 Chrome，Kiwi 浏览器的功能更加完善，尤其是可以安装扩展，尽管移动端对扩展的兼容性并不好。

另外一个可以安装扩展的浏览器是 Yandex 浏览器。但我更喜欢更贴近 Chrome 的 Kiwi 浏览器。记录下我的调整步骤和扩展。

调整

  • 底部工具栏：在 设置-无障碍-底部工具栏，将工具栏位置调整到屏幕下方，方便手指按到。
  • 返回顶部：在设置-无障碍-手形图标，在工具栏中增加一个回到顶部的图标。
  • 修改默认搜索引擎：在搜索栏输入chrome://setting ，在管理扩展引擎中添加喜欢的搜索引擎。
  • 修改主页：在设置-主页 中修改。

扩展

  • floccus
    • 地址： https://chrome.google.com/webstore/detail/floccus/fnaicdffflnofjppbagibeoednhnbjhg?utm_source=chrome-ntp-icon
    • 开源地址： https://github.com/marcelklehr/floccus/
    • 参考教程： http://chromecj.com/accessibility/2019-07/2622.html 移动的设置和桌面端基本是一样的。但是要注意，教程中的目录设置，桌面端可以选择根，但是移动端不能选择根，要选择移动设备书签。否则无法同步。
    • 使用理由：由于大陆的网络问题，实际上并不能随时随地的访问谷歌，而手机端的代理访问并不像桌面端那么智能。即使安装了 Chrome 每次同步书签都需要代理，使用体验并不好。并且 Kiwi 浏览器也并不和 Chrome 的服务器互通。floccus 支持 WebDAV 解决随时同步书签的问题。作为开源软件也更让人信任。
  • Toggle Javascript
    • 地址： https://chrome.google.com/webstore/detail/toggle-javascript/cidlcjdalomndpeagkjpnefhljffbnlo?utm_source=chrome-ntp-icon
    • 使用理由：一些网站用的 Javascript 生成广告，而内容是静态的。禁用 Javascript 就可以屏蔽这样的广告，但是每次修改 设置-网站设置-允许Javascript 是十分麻烦的。而这个插件只需要点击插件图标就可以修改。
  • Reader Mode
    • 地址： https://chrome.google.com/webstore/detail/llimhhconnjiflfimocjggfjdlmlhblm
    • 使用理由：支持移动端的阅读模式。
  • UBlock Origin
    • 地址： https://chrome.google.com/webstore/detail/cjpalhdlnbpafiamejdnhcphjbkeiagm
    • 使用理由：广告屏蔽插件。

问题

  • 视频不能全屏。

总结

并油猴插件的原因在于，我很少在移动端进行复杂的操作。所以只安装了方便阅读文字的插件，以及同步书签的功能。
